In order to link an existing GAR registry from the dashboard, navigate to the **Integrations** tab and select **Docker Registry**. Then, select **Add a Registry** in top right, and select **Google Artifact Registry**. Give the registry a name, for example `my-awesome-registry`:

![GAR](https://imagedelivery.net/l4LYM_vOYKe7O1NCT_Nc_g/b85524a6-0329-4879-c0bb-04f27d140700/large)

Next, input the GCP credentials that you downloaded above, and enter your GAR region.

Then, click **Save Settings**.
