In order to link an existing GCR registry from the dashboard, navigate to the **Integrations** tab and select **Docker Registry**. Then, select **Add a Registry** in top right, and select **Google Container Registry**. Give the registry a name, for example `my-awesome-registry`:

![GCR](https://imagedelivery.net/l4LYM_vOYKe7O1NCT_Nc_g/b85524a6-0329-4879-c0bb-04f27d140700/large)

Next, input the GCP credentials that you downloaded above, and enter your registry URL, the the form `[GCR_DOMAIN]/[GCP_PROJECT_ID]`. For most, the GCR domain will be `gcr.io` (for more information, [click here](https://cloud.google.com/container-registry/docs/overview#registries)).

Then, click **Save Settings**.
