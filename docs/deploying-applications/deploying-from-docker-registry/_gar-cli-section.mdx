Run the following command on the Porter CLI:

```sh
porter connect gar
```

You will be prompted for the full path to the service account key file that you just downloaded. **Enter the full path (not relative) to the key file location**:

```sh
Please provide the full path to a service account key file.
Key file location:
```

Next, you will be prompted to enter the GAR region you want to connect to.:

```sh
Please enter the artifact registry region. For example, us-central-1.
Artifact registry region:
```

Finally, you will be prompted to provide a name for this new registry.

```sh
Give this registry a name:
```

That's it! If you navigate to the "Launch" tab in the dashboard, you should see your existing GAR images in the "Registry" section.
