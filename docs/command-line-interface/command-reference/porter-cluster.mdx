:::info Prerequisites

- You've logged in to the Porter CLI after running [`porter auth login`](./porter-auth)
- You're connected to the correct project by running [`porter projects list`](./porter-project), and set the project via [`porter config set-project [PROJECT_ID]`](./porter-config)

:::

<br></br>

`porter cluster` contains commands for reading from a connected cluster.

### `porter cluster list`

This lists all clusters under a certain project.

### `porter cluster namespace list`

This lists all namespace in a cluster. This is a good mechanism to check that you are properly authenticated to a cluster.
