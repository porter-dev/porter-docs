| heroku addons:destroy | [porter delete addon](./command-reference/porter-delete#porter-delete-addon) |
| heroku apps | [porter list applications](./command-reference/porter-list#porter-list-applications)  |
| heroku apps:create | [porter create](./command-reference/porter-create) |
| heroku apps:destroy | [porter delete app](./command-reference/porter-delete) |
| heroku auth:login | [porter auth login](./command-reference/porter-auth) |
| heroku auth:logout | [porter auth logout](./command-reference/porter-auth) |
| heroku auth:token | [porter config](./command-reference/porter-config) |
| heroku config:set | [porter update env-group set](./command-reference/porter-update) |
| heroku config:set | [porter update env-group unset](./command-reference/porter-update) |
| heroku logs | [porter logs](./command-reference/porter-logs) |
| heroku ps:forward | [porter port-forward](./command-reference/porter-port-forward) |
| heroku run | [porter run -e](./command-reference/porter-run) |
